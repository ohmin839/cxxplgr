# cxxplgr
**cxxplgr** is a text processor for texts in polytonic Greek.\
**CAUTION: this is a work in progress.**

## cxxplgrconv
```
$ echo ">anthr^opos" | plgrconv
ἄνθρωπος
```

## cxxplgrcoll
```
$ cat alpha.txt | plgrconv | plgrcoll
Πάντες
ἄνθρωποι
τοῦ
εἰδέναι
ὀρέγονται
φύσει
σημεῖον
δ'
...
```
